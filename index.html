Project Experience

AI Podcast Generator

Currently building a full stack AI application that allows users to create, view, search & delete AI generated podcasts. Developed front-end using Next.js, TypeScript, Tailwind & Shadcn for better UI/UX. Used ConvexDB for CRUD operations & dynamic data rendering, incorporating secure user authentication. StoryBooks

Built an application where users could post, edit and delete their public and private stories. Configured CRUD operations for backend API with Node and Express.js to enable user interactions & data filtering. Implemented login using passport.js and added Google login features using OAuth.

Huffman File Compression Program

Implemented a lossless file compression program using Huffman encoding algorithm for optimal rates & efficiency. Achieved 80%+ size reduction across multiple formats, optimizing user storage significantly. Processed diverse file types (including .txt & .jpg files), affirming the versatility of the algorithm.

Used Google Firebase backend to store the menu of food choices and fetch data when the user orders food.

Built a LinkedIn clone to allow users to create, and update user accounts, posts etc. Developed the front-end using React, Javascript, and Material-UI for better UI/UX. Utilized Firebase for dynamic data rendering from Firestore (NoSQL), incorporating secure user authentication.

Research on StyleGAN2 ADA Models art generation

Worked with Professor Yuhao Zhu on how StyleGAN2 ADA models learn & predict artistic styles from images of Chinese paintings. Wrote Python scripts to curate a data set of ~3000 paintings from APIs like MET. Developed an image preprocessing technique to standardize training images using aspect ratios, boosting training efficiency by 30%. Performed image filtering through OpenAI's CLIP to collect similar Chinese paintings.
